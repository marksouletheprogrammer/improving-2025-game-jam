# T-Rex Runner Game

A classic endless runner game built with HTML5 Canvas, CSS3, and JavaScript.

## Features

- **Endless Runner Gameplay**: Jump over cacti obstacles as a T-Rex dinosaur
- **Progressive Difficulty**: Game speed increases as your score gets higher
- **Multiple Game States**: Start screen, playing, pause, and game over states
- **Smooth Physics**: Realistic jumping mechanics with gravity
- **Visual Polish**: Scrolling ground, animated clouds, and detailed sprites
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Elegant glassmorphism design with smooth animations

## Controls

- **SPACEBAR**: Jump / Start Game / Restart Game
- **P Key**: Pause/Resume (during gameplay)
- **Start Button**: Begin the game
- **Pause Button**: Pause/Resume the game
- **Restart Button**: Restart after game over

## Game Mechanics

- **Scoring**: Points increase over time as you survive longer
- **Difficulty**: Game speed increases every 100 points
- **Physics**: Realistic gravity and jump mechanics
- **Collision**: Precise collision detection with obstacle hitboxes

## Technical Details

- **HTML5 Canvas**: 2D rendering with smooth animations
- **CSS3**: Modern styling with gradients, blur effects, and animations
- **JavaScript**: ES6 classes and modern web APIs
- **No Dependencies**: Pure vanilla JavaScript implementation

## Running the Game

Simply open `index.html` in any modern web browser. No build process or server required!
